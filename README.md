# ProductPhoneBook
This Java program simulates a product phone book, allowing users to add, view, update, and delete product information including name, price, and rating. It uses a HashMap for efficient storage and retrieval, providing a user-friendly menu-driven interface.
# Each product contains the following information:
Product Name
Product Price
Product Rating (1-5 Stars)
The project uses Java Collections Framework, specifically HashMap, to store and manage product records efficiently.

# Objective
The main objective of this project is to provide a simple product management system where users can:
Add new products
View product details
Display all products
Update product prices
Update product ratings
Delete products
Exit the application

# This project demonstrates the practical implementation of:
Object-Oriented Programming (OOP)
Java Collections Framework
HashMap Data Structure
CRUD Operations
User Input Handling using Scanner
